# 🏥 Hospital & Healthcare Analysis Dashboard

## 📌 Project Overview

This project is an interactive Hospital & Healthcare Analysis Dashboard developed using Power BI. It analyzes hospital operations including patient admissions, doctor information, department revenue, insurance coverage, bed occupancy, emergency cases, and patient feedback.

The dashboard provides business insights through KPI cards, interactive slicers, and multiple visualizations to support data-driven decision-making.

---

## 🎯 Objectives

- Analyze hospital operational data.
- Monitor department-wise revenue.
- Track patient admissions.
- Analyze insurance coverage.
- Monitor bed occupancy.
- Visualize emergency case priorities.
- Create an interactive Power BI dashboard without DAX measures.
